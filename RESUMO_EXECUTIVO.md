## 🚀 Como Começar

### 3 Passos Rápidos

```bash
# 1. Instale
pip install -r requirements.txt

# 2. Use
python src/gerador_manual.py input.json output.docx

# 3. Abra no Word
# Arquivo .docx gerado com padrão profissional!
```

- ⚫ Capa simples
- ⚫ Sem breadcrumbs
- ⚫ Sem legendas
- ⚫ Rodapé básico
- ⚫ Observações em "Obs1:", "Obs2:"
- ⚫ Sem ícones documentados

- ✅ Capa profissional com logo e metadados
- ✅ Breadcrumbs automáticos em cada seção
- ✅ Legendas numeradas hierarquicamente

- ✅ Observações com bullets elegantes
